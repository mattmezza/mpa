Every active context, by channel · user · chat. For developers: open Last-sent payload to see exactly what was assembled and sent to the model — system prompt/identity, history window, and tool definitions — the last time this context ran.

Bind a persona to a chat to override the global active persona there; the change applies on the next turn. Payloads are captured in memory while the agent runs, so a context shows nothing until it has had a turn since startup.
