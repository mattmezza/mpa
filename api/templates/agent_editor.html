Telegram bot

Each agent talks to Telegram through its own bot. Paste a token to bring this agent online; leave it blank and the agent has no bot of its own. The default agent's bot is the one people reach when no other bot or per-chat binding matches. Restart the agent after changing a token.

How do I get a token / find IDs?
  • Bot token — message @BotFather, send /newbot, follow the prompts, copy the token it gives you.
  • User ID — message @userinfobot; it replies with your numeric ID.
  • Chat / group ID — add the bot to the group and send a message; the chat shows up under Telegram chat settings below once it is active.

Only these Telegram users can make this bot reply. Blank = no restriction — set your own ID to lock the bot to you.

Let this bot share a Telegram group with other agent-bots without them talking over each other. Requires the bot's privacy mode off (@BotFather → /setprivacy) so it can see the whole conversation.
