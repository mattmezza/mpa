## Design Guidance

- **Style**: Dark terminal aesthetic, consistent with the website (#060606 bg, #6c9 green accent)
- **Favicon**: A stylized "h" or terminal prompt symbol `>` or a multiplexer icon
- **OG Image**: "humux — Human Multiplexer" text centered, dark background, terminal-style border, green accent
